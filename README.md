# 🌌 Acerca de MovieFlix

**MovieFlix** es una plataforma web enfocada en mostrar películas y contenido multimedia utilizando la API de TMDB con integración personalizada.

El proyecto cuenta con una interfaz moderna inspirada en plataformas de streaming populares y está optimizado para despliegue rápido usando Vercel.

La plataforma permite:

- 🎬 Explorar películas populares
- 🔥 Ver contenido trending
- 📺 Navegar entre categorías
- 🎥 Visualizar posters dinámicos
- ⚡ Renderizado rápido
- 📱 Diseño responsive
- ☁️ Deploy automático en Vercel

El proyecto fue desarrollado para practicar:

- Next.js
- Integración de APIs
- Frontend moderno
- Streaming UI
- Responsive Design
- Deploy Cloud

# ⚡ Instalación

## 1️⃣ Clonar repositorio

```bash
git clone https://github.com/isairey/movieflix
```

---

## 2️⃣ Instalar dependencias

```bash
npm install
```

---

## 3️⃣ Configurar variables de entorno

Crear archivo:

```bash
.env.local
```

Agregar:

```env
TMDB_API_KEY=TU_API_KEY
```

---

## 4️⃣ Ejecutar proyecto

```bash
npm run dev
```
